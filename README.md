# Todos - 待办事项

一个简洁高效的桌面待办事项管理工具，基于 **PySide6** 开发，支持任务栏进度条（Windows）、批量操作和本地 JSON 数据持久化。

## ✨ 特性

- **任务管理**：添加、编辑、删除任务，支持批量选择操作。
- **完成状态**：一键标记任务为已完成/未完成，已完成任务自动置灰并显示 ✅ 前缀。
- **进度追踪**：实时显示任务完成进度条，并在 Windows 任务栏同步显示进度（需 `QtWinExtras` 支持）。
- **数据持久化**：所有任务自动保存至 `todos.json` 文件，下次启动自动加载。
- **批量操作**：支持多选任务，批量标记完成/未完成、批量删除。
- **清空所有**：一键清空所有任务（需二次确认，防止误删）。

## 🛠️ 依赖

- Python 3.6 或更高版本
- [PySide6](https://pypi.org/project/PySide6/)（Qt for Python）

如需任务栏进度条支持（仅 Windows），还需安装 `PySide6.QtWinExtras`（PySide6 自带，无需额外安装，但仅在 Windows 下生效）。

## 📦 安装

1. 克隆仓库或下载 `main.pyw` 文件。
2. 安装依赖：
```bash
pip install PySide6
```

（若使用 Windows 并希望任务栏进度条生效，无需额外操作，PySide6 已包含所需模块。）

## 🚀 使用方法

直接运行 main.pyw：

```bash
python main.pyw
```

或者在文件管理器中双击运行（Windows 下 .pyw 后缀会隐藏控制台窗口）。

### 基本操作

- 添加任务：在顶部输入框输入任务内容，按下回车或点击“添加任务”按钮。
- 选择任务：点击任务列表中的项可单选，按住 Ctrl 或 Shift 可多选。
- 标记完成/未完成：选中任务后，点击“标记完成”或“标记未完成”按钮。
- 编辑任务：选中单个任务，点击“编辑任务”按钮，在弹出的对话框中修改内容。
- 删除任务：选中一个或多个任务，点击“删除任务”按钮，确认后删除。
- 清空所有：点击“清空所有”按钮，确认后删除全部任务。

### 📁 数据存储

所有任务数据保存在程序同目录下的 todos.json 文件中，格式如下：

```json
[
  {"text": "学习 Python", "completed": false},
  {"text": "写 README", "completed": true}
]
```

若文件损坏或不存在，程序将自动创建新的空列表。

## ⚠️ 注意事项

- 任务栏进度条功能仅支持 Windows 系统，且需保证 PySide6 正确安装。在其他操作系统下，该功能自动禁用，不影响正常使用。
- 请确保程序对当前目录有读写权限，否则无法保存数据。
